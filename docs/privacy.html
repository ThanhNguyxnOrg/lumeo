Lumeo — Privacy Policy

Last updated: 2026-05-09

TL;DR. Lumeo does not collect, store, or sell any personal data. API keys you enter (Kyma, Gemini, OpenRouter, Groq, Hugging Face, OpenAI, Google Cloud, LibreTranslate, Soniox) stay on your own device. Subtitle text or audio you choose to translate is sent directly to the provider you pick, for the sole purpose of producing the translation, then discarded. We have no servers, no analytics, no tracking.

What Lumeo is

Lumeo is a Chrome extension that adds bilingual captions and live AI voice-over to YouTube videos. It offers three tiers — pick whichever fits your needs and budget. The extension runs entirely in your browser; there is no Lumeo-operated backend.

What data is processed, by tier

Caption tier (free, BYOK optional)

  • YouTube subtitle text. Lumeo reads the captions YouTube already publishes for the video (or, if none exist and you provide a Soniox key, transcribes the tab audio you share via getDisplayMedia). The text is sent only to the translate provider you choose: Google Translate (free/no key), Gemini, OpenRouter, Groq, OpenAI, Google Cloud Translation, or LibreTranslate.
  • API keys (optional). If you opt into Gemini, OpenRouter, Groq, Hugging Face, OpenAI, Google Cloud, LibreTranslate, Soniox, or Google Cloud Text-to-Speech, the keys you provide stay in chrome.storage.local and are sent only to those providers as the Authorization header or provider-specific key parameter.
  • Per-video subtitle cache. The translated text for each video you process is stored locally so you don't pay to re-translate on revisit. Wipe it from the panel's clear-cache button.

Standard tier (Kyma key required)

  • YouTube audio. When you press Start, Lumeo captures audio from the active video via HTMLMediaElement.captureStream(), splits it into ~5-second chunks, and uploads each chunk to Kyma which routes through Whisper (transcription) → Gemini (translation) → MiniMax (text-to-speech). The translated voice plays back over the original audio. Lumeo itself stores no audio.
  • Kyma API key. Stored locally with TRUSTED_CONTEXTS access level so that page scripts on youtube.com cannot read it.

Realtime tier (Kyma key required)

  • YouTube audio. Captured the same way and streamed peer-to-peer over WebRTC directly to OpenAI Realtime, after Kyma mints a short-lived ephemeral token. Sub-second latency. Audio does not pass through any Lumeo-operated server.
  • Kyma API key. Same handling as Standard tier. The Kyma key itself is never sent to OpenAI; only the ephemeral token is.

Common to all tiers

  • Overlay layout. Position, size, and panel preferences stored in localStorage for the youtube.com origin. Cosmetic only.
  • Settings. Tier, language, voice, volume, and subtitle-style preferences stored in chrome.storage.local. Never leave your device.

What data is NOT collected

  • No telemetry, analytics, crash reports, or usage statistics.
  • No browsing history, video titles, URLs, or YouTube account information are sent anywhere.
  • No personal identifiers, IP logging, cookies, or fingerprints.
  • No advertising, ad targeting, or third-party trackers.

Third-party processors

The provider that processes your data depends entirely on which tier and which translate / TTS / STT provider you select in the panel:

  • Kyma API (api.kymaapi.com) — gateway for the Standard and Realtime tiers. Bills your Kyma balance. See Kyma's privacy policy.
  • OpenAI (api.openai.com) — Realtime tier (peer-to-peer audio); also chat completions for the OpenAI translate option in the Caption tier when you supply an OpenAI key.
  • Gemini (generativelanguage.googleapis.com) — used only if you provide a Gemini API key and choose Gemini translation.
  • OpenRouter (openrouter.ai) — used only if you provide an OpenRouter key and choose OpenRouter translation, including the free model router when available.
  • Groq (api.groq.com) — used only if you provide a Groq key and choose Groq translation/STT options.
  • Hugging Face (api-inference.huggingface.co, huggingface.co) — planned advanced provider option, used only if you provide a Hugging Face token and select it.
  • Google Translate (translate.googleapis.com) — free public endpoint used by the Caption tier's default translate option. Subtitle text only.
  • Google Cloud (translation.googleapis.com, texttospeech.googleapis.com) — used only if you provide a Google Cloud API key and pick the corresponding option.
  • LibreTranslate (libretranslate.com, localhost, or a URL you provide) — used only if you choose LibreTranslate translation.
  • Soniox (stt-rt.soniox.com) — used only if you provide a Soniox key and the video has no captions, in which case Lumeo transcribes shared tab audio in real time.

Each provider processes your data per its own privacy terms. Lumeo is not a party to those terms — your account and direct relationship with each provider are.

Data retention

Lumeo retains nothing on its side. Provider-side retention follows each provider's policy; for transcription, translation, and TTS the data is processed in-memory and not retained. Sessions on the dub tiers automatically end after 60 minutes; closing the tab also ends the session.

Permissions

  • activeTab, scripting — to inject the overlay panel into the YouTube tab when you press Start.
  • storage — to remember your settings and any keys you save across sessions.
  • Host permissions for youtube.com, api.kymaapi.com, api.openai.com, translate.googleapis.com, translation.googleapis.com, texttospeech.googleapis.com, stt-rt.soniox.com — to read YouTube subtitles, capture YouTube audio, and reach the AI providers listed above.

Your rights and choices

  • Stop a session at any time with the Stop button or by closing the tab.
  • Remove any key from the popup or panel at any time, or uninstall the extension to wipe all locally stored data.
  • Use the Caption tier with the free Google Translate option without providing any account or key.

Children

Lumeo is not directed at children under 13. The extension processes no personal information that would identify a user of any age.

Changes to this policy

If we materially change how Lumeo handles data we will update this page and bump the date at the top. Continued use after a change constitutes acceptance.
